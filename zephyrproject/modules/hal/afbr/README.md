# Zephyr HAL - AFBR

This repository is a HAL module supporting the AFBR-S50 Time-of-Flight sensors by Broadcom. It has been adapted from Broadcom's repository in order to be compatible with the Zephyr Project.

Link to Broadcom's repository: https://github.com/Broadcom/AFBR-S50-API/tree/main/AFBR-S50.
